# pandas

## 概述

pandas 是 Python 中最主流的数据分析与操作库，专为处理表格化的结构数据而设计。其核心数据结构 [[wiki/concepts/dataframe|DataFrame]] 提供了从数据读写、清洗、变换到统计分析的完整工具链。

## 核心功能

- **数据读写**：支持 CSV、Excel、SQL、JSON、Parquet 等格式
- **数据选择**：灵活的行列切片与条件筛选
- **数据变换**：列运算、groupby 聚合、pivot/melt 重塑
- **文本处理**：通过 `str` 访问器批量操作字符串列
- **时间序列**：内置日期时间索引与时间窗口操作
- **可视化**：集成 Matplotlib 的绑定绘图接口
